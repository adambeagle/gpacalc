  1. Is my class data anonymous?

    Yes. When you first visit this site your browser is assigned a random key which is placed in a cookie. This key is required to view or edit any data you provide. No other data about your computer, location, etc. is examined or retained. Though the site owner may access the database containing class data, there is nothing that could tie any data back to a particular person.

  2. Do I need to re-enter my class data every time I visit?

    To maintain anonimity and simplicity, this site uses a cookie with a random key which is used to retrieve data. The drawback of this system is the cookie is unique to the computer and browser with which you visited the site. To retain your data, always visit from the same computer with the same browser, and be sure you're not in incognito/private browsing mode. The cookie from this site will expire after 6 months.

  3. How can I add more classes to a semester?

    Every time you edit a semester, there will be five empty spaces for new classes.
